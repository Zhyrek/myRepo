FC Gold

Controls:

Q - Switch to "Create water rods" mode
W - Switch to "Create wood rods" mode
E - Switch to "Create gold rods" mode
R - Switch to "Create ghost rods" mode
A - Switch to "Create counterclockwise wheels (pink)" mode
S - Switch to "Create unpowered wheels (blue)" mode
D - Switch to "Create clockwise wheels (yellow)" mode
[Comma] - Switch to "Move Object" mode
[Period] - Switch to "Delete object" mode
[Space] - Start game/reset world

Click, drag, and release to place and move objects, click the object you want to delete
Scroll wheel in/out to zoom
